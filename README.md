
# QA Take-Home Assignment

## 🧩 Overview

You are provided with a buggy Trello-like React application. Your task is to explore the app, identify bugs, write test cases, and optionally create automated tests.

---

## ✅ Tasks

1. **Bug Identification**
   - Find at least 5 bugs.
   - For each bug, include:
     - Title
     - Steps to Reproduce
     - Expected vs Actual Behavior
     - Severity (Low/Medium/High)

2. **Test Case Writing**
   - Write at least 20 test cases.
   - Use any format you prefer (Markdown, table, etc.)
   - Cover happy paths, edge cases, and negative scenarios.

3. **Optional Automation**
   - Automate 1–3 flows using Selenium, Cypress, Playwright, or Testing Library like Jest.
   - Include a `README` under the `/automation/` folder if applicable.

---

## ⚠️ Important

> We want to see your thinking and approach, so **please avoid using LLMs for test case generation or finding test cases**.

---

## 🛠 Project Setup

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm run dev
```

3. Open your browser to: `http://localhost:5173`

---

## 📁 Submission

Please push your completed assignment to a GitHub repo including:

- `test-cases.md` or `test-cases.xlsx` or `bug-report.xlsx`
